# Lecture 3: Convert the code to TypeScript

## Introduction

Now we have an AI model based on Neural Network. We are going to improve our model. So let's convert the project from .js to .ts node. I haven't changed any logic, just added the types. Feel free to go throught the code. Follow to steps to update it in your machine.

## Steps

### Step 1: Pull the changes

Run in terminal
`git pull`

### Step 2: Install dependencies

Run in terminal
`npm i`

### Step 3: Run the code

Run in terminal
`npm run dev`
